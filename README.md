# NodeJS MySQL Upload Images
>Upload And Save To MySQL
### Installing

1. Clone The Project
```
git clone https://github.com/SharpTH/nodejs-upload-images.git
```
2. Install dependencies
```
cd Node.js-Upload-images
npm install
```
