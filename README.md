# NodeJS Mysql Upload Images
>Upload and save to Mysql

### Installing

1. Clone the project
```
git clone https://github.com/SharpTH/Node.js-Upload-images.git
```
2. Install dependencies
```
cd Node.js-Upload-images
npm install
```
